# Подборка путешествий по-еврейски

Красивая веб-страница с подборкой путешествий в разные страны.

## Особенности
- Адаптивный дизайн
- Красивые картинки стран и блюд
- Кликабельные ссылки на билеты и отели
- Оптимизировано для печати в PDF

## Технологии
- HTML5
- CSS3
